ความรู้พื้นฐานเกี่ยวกับระบบควบคุม (Control System)

1. ระบบควบคุมแบบวงเปิด (Open-Loop Control System)
   - เป็นระบบที่ไม่มีการป้อนกลับ (Feedback)
   - Output ไม่มีผลต่อ Input
   - ไม่สามารถแก้ไขความผิดพลาดได้อัตโนมัติ
   - ตัวอย่าง: เครื่องซักผ้า, ไมโครเวฟ, พัดลม
   - ข้อดี: ราคาถูก, ออกแบบง่าย, บำรุงรักษาง่าย
   - ข้อเสีย: ความแม่นยำต่ำ, ไม่สามารถปรับตัวกับการรบกวนได้

2. ระบบควบคุมแบบวงปิด (Closed-Loop Control System)
   - เป็นระบบที่มีการป้อนกลับ (Feedback)
   - Output จะถูกนำกลับมาเปรียบเทียบกับ Input
   - สามารถแก้ไขความผิดพลาดได้อัตโนมัติ
   - ตัวอย่าง: ระบบครูซคอนโทรลในรถยนต์, ระบบปรับอุณหภูมิห้อง
   - ข้อดี: ความแม่นยำสูง, ปรับตัวกับการรบกวนได้
   - ข้อเสีย: ราคาแพง, ซับซ้อน, อาจเกิดความไม่เสถียร

3. ฟังก์ชันถ่ายโอน (Transfer Function)
   - G(s) = Y(s) / X(s)
   - แสดงความสัมพันธ์ระหว่าง Input และ Output ในโดเมน s
   - ใช้ Laplace Transform
   - ตัวอย่าง: G(s) = K / (τs + 1) สำหรับระบบอันดับหนึ่ง

4. PID Controller
   - P (Proportional): ควบคุมตามสัดส่วนของ Error
   - I (Integral): ขจัดความคลาดเคลื่อนในสถานะคงตัว (Steady-State Error)
   - D (Derivative): ปรับปรุงการตอบสนองชั่วครู่ (Transient Response)
   - สมการ: u(t) = Kp*e(t) + Ki*∫e(t)dt + Kd*de(t)/dt

5. เสถียรภาพ (Stability)
   - ระบบเสถียร: Output มีค่าจำกัดเมื่อเวลาผ่านไป
   - เกณฑ์ Routh-Hurwitz: ใช้ตรวจสอบเสถียรภาพ
   - Root Locus: แสดงตำแหน่งขั้วเมื่อ K เปลี่ยนแปลง
   - Bode Plot: วิเคราะห์เสถียรภาพในโดเมนความถี่

6. การตอบสนอง (Response)
   - Step Response: การตอบสนองต่อสัญญาณขั้นบันได
   - Overshoot: การกระโดดเกินค่าที่ต้องการ
   - Settling Time: เวลาที่ใช้เข้าสู่ค่าที่ต้องการ
   - Rise Time: เวลาที่ใช้เพิ่มจาก 0 ถึง 100%

7. State Space Model
   - ẋ = Ax + Bu
   - y = Cx + Du
   - x = state vector
   - u = input vector
   - y = output vector
   - A, B, C, D = matrices
